# **App Name**: আমার হিসাব (My Accounts)

## Core Features:

- Financial Dashboard: An interactive overview displaying key financial metrics like total income, expenses, and current balance.
- Income Tracking: Record all sources of income with details such as amount, date, and source.
- Expense Tracking: Log all expenses, categorized and tagged with details like amount, date, and purpose.
- Transaction Data Persistence: Securely save all income and expense records permanently in a database for long-term access.
- Financial Reporting: Generate summary reports of income and expenses, categorized for clear financial overview.
- AI-Powered Financial Summaries Tool: A tool to provide intelligent, natural language summaries of spending habits and offer actionable insights.
- Application Settings: Manage user preferences and customizable categories for income and expenses.

## Style Guidelines:

- Primary color: A bold and modern red (#C33B49) for headers, buttons, and highlighting key financial figures, chosen to reflect the explicit user request for 'লাল' (red) while maintaining a contemporary aesthetic. This will also be the color for the top and bottom navigation bars.
- Background color: A very light, subtly tinted beige-pink (#FAEFF0) for main content areas, providing a soft contrast to the primary red and ensuring readability within a light theme.
- Accent color: A vibrant magenta-rose (#DB62AD) to draw attention to interactive elements, alerts, or important notifications, offering a lively and harmonious contrast to the primary red.
- All text will use 'Inter', a neutral and highly readable sans-serif typeface, suitable for both headlines and detailed financial data, ensuring clarity and a modern aesthetic.
- Fixed Top Bar: A prominent top bar displaying the app name 'আমার হিসাব' (My Accounts) and a logo, ensuring constant brand presence.
- Fixed Bottom Navigation Bar: A persistent bottom menu with intuitive icons and labels for Dashboard, My Income, My Expenses, Reports, and Settings, for easy access to core functionalities.
- Clean, line-based icons for all menu items and financial categories, providing clear visual cues without cluttering the interface.
